# Security Testing Portfolio

This repository contains hands-on security testing and vulnerability assessment projects completed during my university coursework.

All work was conducted in isolated virtual machine and container-based lab environments for educational purposes.

---

## Projects

### security-network-attacks
Network attack and reconnaissance laboratory including packet sniffing, spoofing, TCP reset and session hijacking using Scapy and Wireshark.

### security-testing-techniques
Security testing techniques including IDS (Snort), firewall configuration (iptables), encryption analysis (OpenSSL), network scanning (Nmap), and static code analysis (FlawFinder).
